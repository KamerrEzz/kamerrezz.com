## Zone Royale 🎯

**Zone Royale** es un mod para Minecraft 1.20.1 (Forge) que introduce un sistema de zona de combate al estilo *Battle Royale*. Diseñado para servidores multijugador, añade mecánicas de reducción de zona, teletransporte aleatorio y espectador automático, ofreciendo una experiencia competitiva intensa y dinámica.

### ✨ Características
- 🎯 **Zona dinámica** que se reduce con el tiempo.
- 🔀 **Teletransporte aleatorio** para jugadores.
- 👻 **Modo espectador automático** para eliminados.
- 📏 **Escalado automático** según el número de jugadores.
- ⚡ **Sistema completo de comandos** para gestionar partidas.

### 📦 Instalación
1. Instalar **Minecraft Forge 1.20.1-47.4.0** o superior.
2. Descargar `zoneroyale-1.0.0.jar`.
3. Colocar el archivo en la carpeta `mods/` del servidor.
4. Reiniciar el servidor.

### 🎮 Comandos principales
- `/zoneroyale start` — Inicia la partida Battle Royale.
- `/zoneroyale stop` — Detiene la partida.
- `/zoneroyale status` — Muestra el estado actual.
- `/zoneroyale help` — Lista completa de comandos.

### ⚠️ Requisitos
- **Minecraft 1.20.1**
- **Forge 47.4.0+**
- Permisos de operador para usar comandos.
